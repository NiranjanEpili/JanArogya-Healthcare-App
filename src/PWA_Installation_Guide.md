# JanArogya Healthcare App - PWA Installation Guide

## About PWA Features

JanArogya is designed as a Progressive Web App (PWA) to provide:

- **Offline Functionality**: Access healthcare features without internet
- **Voice Navigation**: Navigate using voice commands in local languages
- **Quick Access**: Install on phone home screen like a native app
- **Fast Loading**: Cached resources for instant startup

## Installation Instructions

### On Android/iOS Mobile:
1. Open the app in Chrome/Safari browser
2. Look for "Add to Home Screen" prompt
3. Tap "Add" or "Install"
4. App icon will appear on home screen

### On Desktop:
1. Open in Chrome/Edge browser
2. Click the install icon in address bar
3. Click "Install" in the popup
4. App will open in its own window

## Features Available Offline:
- Emergency contact information
- Basic health assessment
- Medicine reminders
- Voice navigation
- Health tips and information

## Voice Commands (Available in Hindi, English, Punjabi, Bengali):
- "होम जाएं" / "Go home"
- "आपातकाल" / "Emergency"
- "मरीज़ पोर्टल" / "Patient portal"
- "डॉक्टर से मिलें" / "See doctor"

## Troubleshooting:
- If installation fails, try refreshing the page
- Ensure you have a stable internet connection for first install
- Check browser compatibility (Chrome, Safari, Edge recommended)
